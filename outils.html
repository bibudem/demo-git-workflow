Quelques outils intéressants

Git

  • SourceTree: un logiciel graphique pour travailler avec Git et GitHub.
  • Git cheat sheet: il y en a probablement des tonnes du genre, mais celle-ci est intéressante.
  • TortoiseGit: un client du logiciel de gestion de versions Git, implémenté comme une extension shell de Windows.
  • GitBucket: une plateforme de gestion de dépôts de code source alimenté par Scala, extensible et compatible avec l'API GitHub.
